Variables

  • Declared with keyword "var"
  • Initialized with "="
  • E.g. var und; (Undefined)
  • E.g. var float = 4.1;

Numbers

  • Can be written with and without decimal points
  • Numbers with decimal points are floats
  • Examples: var num = 4; var float = 0.2;

Strings

  • A literal type used to define text
  • Unlike in Python, JavaScript has no concept of a "character"
  • A single character is just a string with one character
  • Denoted by "" or ''
  • E.g. var string="a";

Objects

  • Similar to dictionaries in Python
  • They allow you to map keys to values
  • Denoted with {}
  • Keys can be accessed with either .keyname or ["keyname"]
    • Example:
    • var object = {
    • adam: "hello!",
    • zach-key: 4
    • }
