Variables

  • Declared with keyword "var"
  • Initialized with "="
  • E.g. var und; (Undefined)
  • E.g. var float = 4.1;

Numbers

  • Can be written with and without decimal points
  • Numbers with decimal points are floats
  • Examples: var num = 4; var float = 0.2;

Strings

  • A literal type used to define text
  • Unlike in Python, JavaScript has no concept of a "character"
  • A single character is just a string with one character
  • Denoted by "" or ''
  • E.g. var string="a";
  • P.S. Strings are arrays of single characters ;(

Objects

  • Similar to dictionaries in Python
  • They allow you to map keys to values
  • Denoted with {}
  • Keys can be accessed with either .keyname or ["keyname"]
  • Example:
    • var object = {
    • adam: "hello!",
    • zach-key: 4
    • }

Arrays

  • Automatically create keys in numerical order
  • Elements are accessible by their index with [index]
  • E.g: var array = ["YG","Vince Staples","Kendrick Lamar","SOB X RBE"]
  • array[0] == "YG" // True

Number Operators

  • Same ones as Python: +, -, *, /. and %
  • You can combine operators with assignment like in Python too (e.g. +=, -=, *=, /=, %=)
  • E.g. var add = 6 + 9; // equals 15

String Operators

  • + will concatenate two strings (or a string and a non-string)
  • .length will return the number of characters in a string
  • .indexOf(sub) will return the index of the substring
  • Examples:
    • var concat = "hello" + "world"; // helloworld
    • var length = concat.length // 10
    • var indexWorld = concat.indexOf('world'); // 5

Boolean Operators

  • ! will return the opposite of the boolean
  • && will apply "and" to two boolean values
  • || will apply "or" to two boolean values
  • == will check for equality between two values
  • E.g. var dog != false; // True

Array Functions

  • .push(element) appends a value to the end of an array
  • .pop() removes the last element in an array
  • .length tells you the number of items in an array
  • Examples:
    • var array = [4,2,"dog"];
    • array.push('⇚'); // [4,2,"dog","⇚"]
    • array.pop(); // [4,2,"dog"]
    • array.length; // 3

Conditionals

  • They work the same way as in Python
  • Instead of "elif", though, you use "else if"
  • Example:
    • if(answer == "Yes"){
    • // Code
    • } else if(answer == "Maybe"){
    • // Code
    • } else{
    • // Code
    • }
  • P.S. if("string"){} // "string" reads as True since it's a non-empty string

For Loops

  • Requires 3 statements: for (s1, s2, s3)
  • The first statement is run before the loop starts
  • The second statement is checked to decide whether the code should loop again
  • The third statement is run after each loop
  • Example:
    • for (var i = 0; i < 10; i++){
    • console.log("HORSE!");
    • //this for loop will log "HORSE!" to the console 10 times
    • }

While Loops

  • Needs one statement: while(s1)
  • That statement is run before every loop to see if the code should be run
  • E.g.: while(some_var = true){console.log("Mxyzptlk!")}

Functions

  • Almost exactly like Python functions
  • Function names are optional in JavaScript
  • Functions can be saved to a variable
  • Functions take many inputs and can return one output
  • E.g. function myFunc(){return 4;} // Returns 4
  • E.g. var andy = function(arg1, arg2){return arg1 + arg2;}
